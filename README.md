# Prototype — Module Thông báo (CMS Cổng Pháp luật Quốc gia)

Prototype giao diện quản trị thông báo theo phong cách Directus, bám mô hình dữ liệu thật.
Trang HTML self-contained (CSS/JS nhúng sẵn, không phụ thuộc file ngoài).

## Xem
- Mở trực tiếp `index.html` bằng trình duyệt, **hoặc**
- Truy cập bản GitHub Pages sau khi bật (xem hướng dẫn bên dưới).

## Nội dung
10 màn: Danh sách thông báo · Tạo & gửi · Loại thông báo · Mẫu · Nhóm & đối tượng ·
Cấu hình gửi · Thiết bị (Push) · Trạng thái gửi · Lịch sử nhận · Thống kê.

> Chỉ là prototype minh hoạ, dữ liệu là mock.
